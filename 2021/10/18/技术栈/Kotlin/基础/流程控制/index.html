条件语句、循环语句、跳转语句

条件语句

if 语句

基本语法

if 作为表达式

if 表达式会返回一个值，所以kotlin 没有三元运算符，使用if 语句就可以达到三元运算符的效果

补充：

  • 如果没有返回值则默认赋值kotlin.Unit；
  • 最为表达式只能包含一个 if和else；
  • 返回的是代码块最后一句的执行结果

when 语句

when 语句类似于 switch 语句

基本语法

when 将依次匹配所有分支，直到找到一个满足条件的分支

when 的参数如果是只读变量，它可以直接在括号内声明，此变量的作用域就是when内

分支条件

分支条件可以为任意表达式，对比的只是表达式的结果

可以使用 in 或 !in 匹配一个范围或集合

可以使用 is 或 !is 匹配特定类型，这里存在只能强制转换，在匹配到的代码块中可以直接访问该类型的方法和属性

when 作为表达式

when 作为表达式会返回第一个匹配到分支最后一句的结果

并且一般情况下else是必须存在的，除非编译器能证明所有可能的条件都包含（例如，将enum作为参数）

when 表达式中以下情况下 else 是必须的 ：

  • 参数为Boolean、enum 或sealed类型，或者它们为空的同类；
  • when 不能覆盖所有可能的条件；

要为多个匹配项共用一个代码块

when 可以不填写参数，此时可以用来代替 if 语句，这里分支条件结果必须是布尔类型

循环语句

for 语句

for 语句主要是用来遍历各种集合数组等数据

基本语法

for 循环可以变例任何可以提供迭代器的对象：

  • 具有一个成员或 iterator() 扩展方法返回 Iterator<>；
  • 有成员或 next()扩展方法；
  • 有成员或hasNext()扩展方法，返回布尔类型

迭代数字

根据索引遍历数组

根据索引遍历数组，此时不会创建迭代器对象

while 和 do-while 循环

while 和 do-while 循环都是在满足条件时执行其注意，区别只是条件检查的时间

跳转语句

跳转表达式有三个：

  1. break：结束最近的循环；
  2. continue：跳过本轮循环进入下一次循环；
  3. return：从最近的函数或匿名函数返回

标签

kotlin 中的任何表达式都可以被标记，形式为标识符后加 @，标签可以配合break、continue和return使用
