Calendar（Java1.1引入）

Calendar类可以看作是对Date类的一个升级，它将日期分为多个部分：年、月、日、时、分、秒、星期等，添加了对每个部分进行运算的功能。

Date 和 Calendar 的局限性

  • 可变性：日期和时间这样的类是不应该被随意修改的；
  • 偏移性：Date中的年份是从1900开始的，而Date中的月份都从0开始，不利于表示；
  • 格式化：Calendar类没有提供格式化日历的方法；
  • 线程安全性：Date和Calendar类都不是线程安全的，也不能处理闰秒等问题。
